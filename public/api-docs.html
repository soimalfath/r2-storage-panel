POST Upload Single File
/api/upload

Upload a single file. Images are auto-converted to WebP.

Request

curl -X POST \
  -H "X-API-Key: your-api-key" \
  -H "X-Bucket-ID: bucket-id" \
  -F "file=@image.jpg" \
  http://localhost:3000/api/upload

Response

{
  "meta": { "code": 200, "status": "success" },
  "data": {
    "filename": "1640995200-image.webp",
    "key": "1640995200-image.webp",
    "size": 245760,
    "contentType": "image/webp",
    "publicUrl": "https://cdn.example.com/...",
    "downloadUrl": "/r2/download/..."
  }
}
POST Upload Multiple Files
/api/upload-multiple

Upload up to 10 files at once.

Request

curl -X POST \
  -H "X-API-Key: your-api-key" \
  -H "X-Bucket-ID: bucket-id" \
  -F "files=@file1.jpg" \
  -F "files=@file2.png" \
  http://localhost:3000/api/upload-multiple

Response

{
  "meta": { "code": 200, "status": "success" },
  "data": {
    "uploaded": [...],
    "errors": [],
    "summary": {
      "total": 2, "successful": 2, "failed": 0
    }
  }
}
GET List Files
/api/files

List files with pagination. Query params: limit token prefix

Request

curl -H "X-API-Key: your-api-key" \
  -H "X-Bucket-ID: bucket-id" \
  "http://localhost:3000/api/files?limit=50"

Response

{
  "meta": { "code": 200, "status": "success" },
  "data": {
    "files": [
      {
        "key": "image.webp",
        "size": 245760,
        "publicUrl": "https://...",
        "presignedUrl": "https://...",
        "downloadUrl": "/r2/download/..."
      }
    ],
    "pagination": { "nextToken": null }
  }
}
DELETE Delete File
/api/files/:key

Delete a file by its key.

Request

curl -X DELETE \
  -H "X-API-Key: your-api-key" \
  -H "X-Bucket-ID: bucket-id" \
  http://localhost:3000/api/files/image.webp

Response

{
  "meta": { "code": 200, "status": "success",
            "message": "File deleted successfully" },
  "data": { "key": "image.webp" }
}
GET Storage Stats
/api/stats

Get storage usage statistics for the active bucket.

Request

curl -H "X-API-Key: your-api-key" \
  -H "X-Bucket-ID: bucket-id" \
  http://localhost:3000/api/stats

Response

{
  "meta": { "code": 200, "status": "success" },
  "data": {
    "totalSize": 1048576,
    "totalFiles": 42,
    "fileTypes": { "image": 30, "doc": 12 }
  }
}
